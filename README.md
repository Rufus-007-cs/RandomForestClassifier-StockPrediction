It is a sytem that first collects data from resouces about the price history of the stock and performs EDA(exploratory data analysis) and removes outliers and unecessary data from the dataset using data analytics strategies then the program proceeds to use a RandomForestClassifier predict the price , this specific system has given a 57% predicted  output to the actual data ,the future enhancements are to add the parameters that affect the stock price to the system inorder to determine the causes for the increase/decrease  in stock prices to develop the system further.
